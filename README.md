# MAGistral Ingénierie — Site Vitrine

Site vitrine Next.js 14 + TypeScript + Tailwind CSS.

## Pages

- `/` — Accueil
- `/services` — Services d'expertise
- `/a-propos` — Présentation
- `/contact` — Formulaire de contact

## Déploiement sur Vercel

1. Crée un repo GitHub et pousse ce code
2. Va sur vercel.com → "New Project"
3. Importe ton repo GitHub
4. Vercel détecte automatiquement Next.js
5. Clique "Deploy" — c'est gratuit !

## Développement local

```bash
npm install
npm run dev
```

## Domaine personnalisé

Une fois déployé sur Vercel :
1. Va dans ton projet → Settings → Domains
2. Ajoute `magistral-ingenierie.fr`
3. Configure les DNS chez OVH selon les instructions Vercel
